RECIPE PLANNER

This is a meal planner web application. Users can search up recipes. Once they have found a recipe that they like, they can click on the recipe and a new page will display the ingredients and other information about the recipe. If the users like the recipe enough, they can add the ingredients they want to purchase to the grocery list. Also, they can add the recipe to the meal scheduler. Many technologies were used to create this application which includes HTML, CSS, ReactJS, Food2Fork API, and Bootstrap.

ONLINE STORE

This is an online store that functions similarly to that of a real online store. Users can add an item that they like to the cart. They call also remove items from the cart. The application keeps track of which items were added to the cart and how many of each item was added to the cart. If the users want to remove all of a specific item from the cart, then that functionality also exists in the application. The technologies used for this project were HTML, CSS, ReactJS, Redux, and React Router.
